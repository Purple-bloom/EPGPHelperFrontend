# EPGP Helper Frontend
Frontend counterpart to https://github.com/Purple-bloom/EPGPHelper

# Setup
- git clone this project
- navigate to install folder and run "npm install" to install dependencies   
- start the app with a script like "PORT=1234 REACT_APP_BACKEND_URL=yourBackendUrl:10000 npm start" (change to your port and backendUrl) :)
- the "REACT_APP_BACKEND_URL" is the thing you need to update with the port you configured for your backend server in https://github.com/Purple-bloom/EPGPHelper !
